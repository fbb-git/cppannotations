Typos, corrections, etc. for the C++ Annotations version 10.8.1

  • Section 5.2.1: the string constructor string(std::initializer_list<char) is not mentioned. It initializes a std::string object from the characters specified in the initializer_list. Be careful not to use the brace-enclosed constructor for std::string(size_t count, char c), as, e.g., string oops{ 1, 'x' } constructs a string consisting of two characters: ascii-1 and 'x', instead of a string object consisting of one x-character.
  • Section 20.2.2.1: std::thread_local should be thread_local;
